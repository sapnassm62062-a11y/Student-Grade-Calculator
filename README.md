# Student Grade Calculator 🎓

## Project Description

Student Grade Calculator is a Java-based console application that calculates a student's total marks, average percentage, and grade based on marks obtained in multiple subjects.

This project is developed using basic Java concepts like variables, loops, arithmetic operations, and conditional statements.

## Features

- Take marks for multiple subjects (out of 100)
- Calculate total marks
- Calculate average percentage
- Assign grades based on percentage
- Display student result clearly

## Technologies Used

- Java
- Scanner Class
- Object-Oriented Programming Basics

## Grade Criteria

| Percentage | Grade |
|------------|-------|
| 90 - 100   | A+ |
| 80 - 89    | A |
| 70 - 79    | B |
| 60 - 69    | C |
| 50 - 59    | D |
| Below 50   | F |
